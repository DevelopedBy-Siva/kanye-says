# kanye-says

A simple desktop app that serves up random Kanye West quotes for daily inspiration.

### How to run

1. Python used:

   ```bash
   python --version
   Python 3.11.9
   ```

2. Execute the main Python script:
   ```bash
   python main.py
   ```
